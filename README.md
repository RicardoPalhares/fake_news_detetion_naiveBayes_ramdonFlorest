# Fake News Detection using Machine Learning

A Python-based machine learning project that detects and classifies fake news articles using text processing techniques and two distinct classification algorithms: **Naive Bayes** and **Random Forest**.

---

## 🚀 Features
* **Text Preprocessing:** Cleans and normalizes raw text data (lowercasing, punctuation removal, stop-word elimination, and stemming/lemmatization).
* **Feature Extraction:** Converts text data into numerical features using **TF-IDF (Term Frequency-Inverse Document Frequency)** vectorization.
* **Dual Classifier Comparison:** Implements and compares the performance of:
  * **Multinomial Naive Bayes:** A probabilistic classifier ideal for text data.
  * **Random Forest:** An ensemble tree-based classifier that captures complex feature interactions.
* **Evaluation Metrics:** Outputs comprehensive performance metrics including Accuracy, Precision, Recall, F1-Score, and Confusion Matrices.

---

## 🛠️ Tech Stack & Libraries
* **Language:** Python 3.x
* **Core Libraries:**
  * `pandas` & `numpy` (Data manipulation)
  * `scikit-learn` (Machine learning models, feature extraction, evaluation)
  * `nltk` or `spacy` (Natural language processing)
  * `matplotlib` & `seaborn` (Data visualization)
